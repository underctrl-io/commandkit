### cache(fn, params?): F

Wraps an async function with caching capability

```ts
const cachedFetch = cache(async (id: string) => {
  return await db.findOne(id);
}, { ttl: '1h' });
```

| Parameter | Type | Optional | Description |
| ----------- | ----------- | ----------- | ----------- |
| fn | F | ❌ | The async function to cache |
| params | [CacheMetadata](/docs/api-reference/types/CacheMetadata) | ✅ | Optional cache configuration |
