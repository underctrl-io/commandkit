### revalidate(tag, args): [Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)\<T>

Forces a refresh of cached data by its tag (on-demand revalidation)

```ts
const freshData = await revalidate('user:123');
```

| Parameter | Type | Optional | Description |
| ----------- | ----------- | ----------- | ----------- |
| tag | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ❌ | The cache tag to revalidate |
| args | [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)\<any> | ❌ | Arguments to pass to the cached function |
