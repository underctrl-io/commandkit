## CommandKitOptions

Options for instantiating a CommandKit handler.

| Property | Type | Value | Description |
| ----------- | ----------- | ----------- | ----------- |
| cacheProvider | null \| CacheProvider | N/A | The cache provider to use with CommandKit. Defaults to MemoryCache provider.Set this to `null` to not use any cache provider. |
| client | Client | N/A | The Discord.js client object to use with CommandKit. |
